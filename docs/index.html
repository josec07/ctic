CTIC

Detect clip-worthy moments from Twitch chat in real-time

GitHub Download

Quick Start

$ ./ctic add xqc https://twitch.tv/xqc
$ ./ctic run
[THREAD:xqc] Connected to #xqcow
[THREAD:xqc] Detecting moments...
2026-03-01T10:21:06Z matched POG burst=4 users=4

How It Works

CTIC connects to Twitch IRC anonymously (no OAuth needed) and watches for chat bursts — moments when multiple viewers react simultaneously with similar words.

Twitch IRC → Parse → Word Match → Burst Detect → CSV Output

Results are saved to CSV files with timestamps, matched words, burst counts, and sample messages.

Install

$ git clone https://github.com/yourusername/ctic
$ cd ctic
$ make
$ ./ctic --help

Features

  • Zero-setup IRC — Anonymous Twitch connection (no OAuth)
  • Multi-creator monitoring — Watch multiple channels simultaneously
  • Configurable detection — Tune thresholds via JSON profiles
  • Tiered word lists — High, medium, easy, high-negative
  • CSV output — Full metadata for analysis
  • Open source — GPL v3

Configurable

Edit .ctic/profiles/balanced.json to adjust detection sensitivity:

{
  "tiers": {
    "high": {
      "burst_threshold": 3,
      "min_word_length": 3,
      "cooldown_seconds": 60,
      "require_unique_users": 2
    }
  }
}

See tutorial.md for full tuning guide.
